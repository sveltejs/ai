Svelte Counter Component

Here's a simple counter component:

<script>
  let count = 0;
  function increment() {
    count++;
  }
  function decrement() {
    count--;
  }
  function reset() {
    count = 0;
  }
</script>
<div class="counter">
  <h1>Counter: {count}</h1>
  
  <div class="buttons">
    <button on:click={decrement}>Decrement</button>
    <button on:click={increment}>Increment</button>
    <button on:click={reset}>Reset</button>
  </div>
</div>

Alternative: More Compact Version

<script>
  let count = 0;
</script>
<div class="counter">
  <button on:click={() => count--}>-</button>
  <span>{count}</span>
  <button on:click={() => count++}>+</button>
</div>
<style>
  .counter {
    display: flex;
    gap: 1rem;
    align-items: center;
  }

</style>

With Props (Reusable)

<script>
  export let initialValue = 0;
  export let step = 1;
  
  let count = initialValue;
</script>
<div class="counter">
  <button on:click={() => count -= step}>-{step}</button>
  <span>{count}</span>
  <button on:click={() => count += step}>+{step}</button>
</div>

Key Svelte Features Used

  • Reactive variables: let count = 0 automatically updates the DOM
  • Event handlers: on:click directive
  • Scoped styles: CSS is automatically scoped to the component
  • Exported props: export let makes values configurable

Choose the version that best fits your needs! The first is most feature-rich, the second is minimal, and the third is most reusable.
